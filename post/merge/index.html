  • .merge

.merge

조인 키에 기반하여 데이터프레임을 병합
Parameter: on, how, suffixes, left_index, rigth_index, left_on, right_on

  • 데이터프레임 생성
df1 = pd.DataFrame({'key1' : np.arange(3),
                    'key2' : ['LA', 'SF', 'LV'],
                    'value1' : np.random.randn(3)})

df1
key1 key2 value1
0 0 LA -0.701438
1 1 SF 0.754723
2 2 LV 0.277378 
df2 = pd.DataFrame({'key1' : np.arange(4),
                    'key2' : ['LA', 'SF', 'KC', 'NY'],
                    'value1' : np.random.randn(4)})

df2
key1 key2 value1
0 0 LA 0.916051
1 1 SF -0.225716
2 2 KC 0.314048
3 3 NY 1.582054
  • on 파라미터
  • 조인 키로 사용할 컬럼을 지정
pd.merge(df1, df2, on='key1')
key1 key2_x value1_x key2_y value1_y
0 0 LA -0.609754 LA 0.440242
1 1 SF -1.328705 SF 0.543113
2 2 LV 1.212269 KC 0.222636 
pd.merge(df1, df2, on='key2')
key1_x key2 value1_x key1_y value1_y
0 0 LA -0.609754 0 0.440242
1 1 SF -1.328705 1 0.543113
  • how 파라미터
    • SQL에서 사용되는 것과 같은 조인 방식을 고를 수 있다.
    • inner : 키가 일치하는 행만 조인 (default)
    • left : 왼쪽 DF가 기준이 되어 오른쪽 DF에 값이 없으면 NaN으로 채운다. (left outer join)
    • right : 오른쪽 DF가 기준이 되어 왼쪽 DF에 값이 없으면 NaN으로 채운다. (right outer join)
    • outer : 모든 행이 조인되며 해당하는 키의 행에 값이 없으면 NaN으로 치운다. (full outer join)


  • how='left'
pd.merge(df1, df2, on='key2', how='left')
key1_x key2 value1_x key1_y value1_y
0 0 LA -0.701438 0.0 0.916051
1 1 SF 0.754723 1.0 -0.225716
2 2 LV 0.277378 NaN NaN


  • how='right'
pd.merge(df1, df2, on='key2', how='right')
key1_x key2 value1_x key1_y value1_y
0 0.0 LA -0.701438 0 0.916051
1 1.0 SF 0.754723 1 -0.225716
2 NaN KC NaN 2 0.314048
3 NaN NY NaN 3 1.582054


  • how='outer'
pd.merge(df1, df2, on='key2', how='outer')
key1_x key2 value1_x key1_y value1_y
0 0.0 LA -0.701438 0.0 0.916051
1 1.0 SF 0.754723 1.0 -0.225716
2 2.0 LV 0.277378 NaN NaN
3 NaN KC NaN 2.0 0.314048
4 NaN NY NaN 3.0 1.582054


  • suffixes 파라미터
    • 컬럼 원본 DF를 식별하게 해주는 식별자 지정
pd.merge(df1, df2, on='key2', how='outer', suffixes=['_left', '_right'])
key1_left key2 value1_left key1_right value1_right
0 0.0 LA -0.701438 0.0 0.916051
1 1.0 SF 0.754723 1.0 -0.225716
2 2.0 LV 0.277378 NaN NaN
3 NaN KC NaN 2.0 0.314048
4 NaN NY NaN 3.0 1.582054


  • left_index, rigth_index
    • 인덱스를 이용해서 조인할 때 사용하는 파라미터 (Boolean)


  • left_on, right_on
    • 조인하는 컬럼 명을 명시해 줄 때 사용
df11 = df1.set_index('key1')
df11
key2 value1
key1
0 LA -0.701438
1 SF 0.754723
2 LV 0.277378 
df22 = df2.set_index('key1')
df22
key1 key2 value1
0 0 LA 0.916051
1 1 SF -0.225716
2 2 KC 0.314048
3 3 NY 1.582054 
pd.merge(df11, df2, left_index=True, right_index=True)
key2_x value1_x key1 key2_y value1_y
0 LA -0.701438 0 LA 0.916051
1 SF 0.754723 1 SF -0.225716
2 LV 0.277378 2 KC 0.314048 
pd.merge(df11, df2, left_index=True, right_on='key1')
key2_x value1_x key1 key2_y value1_y
0 LA -0.701438 0 LA 0.916051
1 SF 0.754723 1 SF -0.225716
2 LV 0.277378 2 KC 0.314048
